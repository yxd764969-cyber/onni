# Onni 韩语语法本（初+中级）

> 覆盖 TOPIK 1~4 核心语法，按「助词 → 动词变形 → 时态 → 句尾语尾 → 连接语尾 → 高频句型」顺序整理。
> 每条含：意义 · 结构 · 例句 · 常见错误。

---

## Part 1 · 助词（조사）

### 1.1 主语助词 이/가 vs 主题助词 은/는

| 助词 | 收音 | 无收音 | 作用 |
|-----|-----|-------|-----|
| 이 / 가 | 이 | 가 | 强调主语（谁做的） |
| 은 / 는 | 은 | 는 | 强调主题/对比 |

- 저는 학생이에요. (我是学生。) → 介绍/对比
- 제가 했어요. (是我做的。) → 强调「是我」

**易错**：初次自我介绍用 는（我(呢)是…），不用 가。

### 1.2 宾语助词 을/를

- 收音 → 을，无收音 → 를
- 例：커피를 마셔요. / 밥을 먹어요.
- 口语可省略：커피 마셔요.

### 1.3 处所/时间助词 에

- **静态处所**（在哪儿）+ 有 / 是 / 去 / 来：집에 있어요. / 학교에 가요.
- **时间点**：3시에 만나요. (三点见。)
- 例外：오늘/어제/내일 不加 에。

### 1.4 动态处所助词 에서

- **在哪儿做动作**：학교에서 공부해요. (在学校学习。)
- **从哪儿**（起点）：한국에서 왔어요. (来自韩国。)

**对比**：에 = 静态存在/到达点；에서 = 发生动作/出发点。

### 1.5 与人相关：에게 / 한테 / 께

- 에게（书面）＝ 한테（口语）＝ 给/对某人：친구한테 선물을 줬어요.
- 께（对长辈敬语）：부모님께 편지를 썼어요.
- **反向**（从某人那儿）：에게서 / 한테서。

### 1.6 手段/方向 (으)로

- 무엇으로 = 用什么：젓가락으로 먹어요.
- 어디로 = 往哪儿：학교로 가요.
- 收音 → 으로；无收音或 ㄹ收音 → 로。

### 1.7 并列 와/과 · (이)랑 · 하고

- 와/과（书面）；(이)랑（口语，最亲切）；하고（中间）。
- 친구와 밥을 먹어요. / 친구랑 밥 먹어. / 친구하고 밥 먹어요.

### 1.8 其他高频助词

| 助词 | 意义 | 例句 |
|-----|------|-----|
| 도 | 也 | 저도 갈래요. |
| 만 | 只 | 이것만 주세요. |
| 부터 | 从…开始 | 9시부터 시작해요. |
| 까지 | 到…为止 | 5시까지 끝내요. |
| 보다 | 比 | 저보다 커요. |
| 처럼/같이 | 像…一样 | 눈처럼 하얘요. |
| 마다 | 每 | 아침마다 커피를 마셔요. |

## Part 2 · 谓词变形基础

### 2.1 词干 + 语尾

- 原形去掉 `다` = 词干：가다 → 가 / 먹다 → 먹 / 예쁘다 → 예쁘。
- 语尾按功能拼接：가 + 요 = 가요 / 먹 + 어요 = 먹어요。

### 2.2 아/어/여 变化规则

- 词干最后音节元音 **ㅏ, ㅗ** → 接 **아**
- 其他元音 → 接 **어**
- 하다 → 여（缩为 해）

| 原形 | 词干 | +아/어요 |
|-----|-----|---------|
| 가다 | 가 | 가요 (缩合) |
| 오다 | 오 | 와요 (缩合) |
| 먹다 | 먹 | 먹어요 |
| 마시다 | 마시 | 마셔요 (缩合) |
| 공부하다 | 공부하 | 공부해요 |

### 2.3 常见不规则变形

- **ㅂ 불규칙**：ㅂ + 母音 → 우
  - 덥다 (热) → 더워요 / 춥다 → 추워요 / 쉽다 → 쉬워요
- **ㄷ 불규칙**：ㄷ + 母音 → ㄹ
  - 듣다 (听) → 들어요 / 걷다 (走) → 걸어요
- **ㅅ 불규칙**：ㅅ + 母音 → 脱落
  - 짓다 (盖) → 지어요 / 낫다 (好转) → 나아요
- **ㅡ 탈락**：ㅡ + 아/어 → 脱落
  - 예쁘다 → 예뻐요 / 바쁘다 → 바빠요 / 쓰다 → 써요
- **르 불규칙**：르 + 어 → ㄹ러 / ㄹ라
  - 부르다 → 불러요 / 모르다 → 몰라요 / 빠르다 → 빨라요
- **ㅎ 불규칙**（形容词）：
  - 어떻다 → 어때요 / 그렇다 → 그래요 / 하얗다 → 하얘요

### 2.4 三个体系（谚谅：文体等级）

| 文体 | 语尾 | 场合 |
|-----|------|-----|
| 합쇼체 (格式体) | -ㅂ니다/습니다 | 新闻、演讲、职场正式 |
| 해요체 (非格式敬语) | -아요/어요 | 日常礼貌，**默认必学** |
| 해체 (반말) | -아/어 | 朋友、恋人、小孩 |

同一句「我吃饭」三体系写法：
- 밥을 먹습니다. (合쇼)
- 밥을 먹어요. (해요)
- 밥 먹어. (반말)

## Part 3 · 时态

### 3.1 现在时

- 해요体：词干 + 아/어요 → 가요, 먹어요, 해요
- 합쇼体：词干（有收音+습니다 / 无收音+ㅂ니다）→ 먹습니다, 갑니다

### 3.2 过去时 -았/었/였-

- 词干 + 았/었 + 语尾
- 갔어요, 먹었어요, 공부했어요, 예뻤어요

### 3.3 将来时 · 意志/推测

- **-(으)ㄹ 거예요**（一般将来 / 推测）
  - 내일 갈 거예요. (明天会去。)
- **-겠-**（意志 / 推测 / 客套）
  - 제가 하겠습니다. (我来做。) / 알겠습니다. (明白了。)

### 3.4 进行 · 状态

- **-고 있다** = 正在（动作进行）
  - 지금 밥을 먹고 있어요. (正在吃饭。)
- **-아/어 있다** = 处于…状态（结果持续）
  - 앉아 있어요. (坐着。) / 문이 열려 있어요. (门开着。)

## Part 4 · 敬语系统

### 4.1 主语敬语 -(으)시-

- 词干 + (으)시 + 语尾
- 가시다, 드시다, 계시다
- 아버지께서 오세요. (爸爸来。)

### 4.2 特殊敬语词汇替换

| 一般 | 敬语 | 意义 |
|-----|------|-----|
| 먹다/마시다 | 드시다 / 잡수시다 | 吃/喝 |
| 자다 | 주무시다 | 睡 |
| 있다 (在) | 계시다 | 在 |
| 있다 (有) | 있으시다 | 有 |
| 죽다 | 돌아가시다 | 去世 |
| 말하다 | 말씀하시다 | 说 |
| 주다 | 드리다 | 给 |
| 만나다 | 뵙다 | 见 |
| 집 | 댁 | 家 |
| 이름 | 성함 | 姓名 |
| 나이 | 연세 | 年龄 |
| 생일 | 생신 | 生日 |
| 밥 | 진지 | 饭 |

### 4.3 自谦

- 저 (我，谦称) / 저희 (我们)
- 드리다 (给的谦语) / 여쭙다 (问的谦语)

## Part 5 · 常用句尾

| 语尾 | 意义 | 例句 |
|-----|------|-----|
| -아/어요 | 陈述/疑问/命令 (해요体) | 밥 먹어요. |
| -(으)세요 | 请…（礼貌请求） | 앉으세요. |
| -지 마세요 | 请不要… | 걱정하지 마세요. |
| -(으)ㄹ까요? | ...呢？/要不要..？ | 뭐 먹을까요? |
| -(으)ㅂ시다 | 一起…吧 | 갑시다. |
| -네요 | 感叹（新发现） | 예쁘네요! |
| -군요/는군요 | 感叹 | 정말 맛있군요. |
| -죠 (지요) | 是吧/对吧 | 맞죠? |
| -잖아요 | 你也知道嘛 | 오늘 비 오잖아요. |
| -거든요 | 因为…嘛 | 좀 바쁘거든요. |

## Part 6 · 连接语尾

### 6.1 并列 / 顺承

- **-고**：又…又…/然后
  - 밥을 먹고 커피를 마셨어요.
- **-아/어서**：因为/然后（前后紧密相关）
  - 배가 고파서 밥을 먹었어요. (因为饿了所以吃了饭。)

### 6.2 原因

- **-아/어서** vs **-(으)니까**
  - 아/어서：客观原因，**后不能接命令/劝诱**
  - (으)니까：主观理由，**可接命令/劝诱**
  - 비가 오니까 우산 가져가세요. ✅
  - 비가 와서 우산 가져가세요. ❌

- **-기 때문에**：书面/较强因果
  - 시간이 없기 때문에 못 가요.

### 6.3 转折

- **-지만**：但是
  - 비싸지만 맛있어요.
- **-(으)ㄴ데/는데**：转折/背景铺垫（超高频！）
  - 지금 바쁜데 나중에 얘기해요.

### 6.4 条件 / 假设

- **-(으)면**：如果
  - 시간 있으면 같이 가요.
- **-았/었으면 좋겠다**：要是…就好了
  - 돈이 많았으면 좋겠어요.

### 6.5 目的

- **-(으)러 가다/오다**：去/来做…（动作目的）
  - 밥을 먹으러 왔어요.
- **-(으)려고**：为了…（意图）
  - 한국어를 배우려고 한국에 왔어요.
- **-기 위해서**：为了…（书面/正式）

### 6.6 时间

- **-(으)ㄴ 후에** / **-고 나서**：…之后
- **-기 전에**：…之前
- **-(으)면서**：一边…一边
- **-자마자**：一…就
- **-는 동안**：…期间
- **-(으)ㄹ 때**：…的时候

## Part 7 · 高频句型（背这些就够用）

### 7.1 请求 / 服务场景

- **N + 주세요** = 请给我 N（**点单万能句**）
  - 아이스 아메리카노 한 잔 주세요.
- **V-아/어 주세요** = 请为我 V
  - 사진 좀 찍어 주세요.
- **N + 있어요?** = 有 N 吗？
- **N + 얼마예요?** = N 多少钱？

### 7.2 意愿 / 能力

- **-고 싶다** = 想要
  - 김치찌개를 먹고 싶어요.
- **-(으)ㄹ 수 있다/없다** = 能/不能
  - 한국말을 할 수 있어요.
- **-(으)ㄹ 줄 알다/모르다** = 会/不会（技能）
  - 운전할 줄 알아요.

### 7.3 义务 / 许可

- **-아/어야 하다/되다** = 必须
  - 지금 가야 해요.
- **-아/어도 되다** = 可以…
  - 여기 앉아도 돼요?
- **-(으)면 안 되다** = 不可以
  - 여기서 담배 피우면 안 돼요.

### 7.4 尝试 / 经验

- **-아/어 보다** = 试着…
  - 이거 한번 먹어 보세요.
- **-(으)ㄴ 적이 있다/없다** = 曾经…过
  - 한국에 가 본 적이 있어요.

### 7.5 推测

- **-(으)ㄹ 것 같다** = 好像会…（对未来）
- **-(으)ㄴ/는 것 같다** = 好像…（对现在/过去）
  - 비가 올 것 같아요. (好像要下雨。)
  - 저 사람 한국 사람인 것 같아요. (那人好像是韩国人。)

### 7.6 决定 / 计划

- **-기로 하다** = 决定要…
  - 내일 만나기로 했어요.
- **-(으)ㄹ게요** = 我会…（对听者的承诺）
  - 제가 갈게요.

## Part 8 · 数字 · 量词（点单/购物必背）

### 8.1 两套数字

| 汉字词数字 | 固有词数字 |
|-----------|-----------|
| 일 이 삼 사 오 육 칠 팔 구 십 | 하나 둘 셋 넷 다섯 여섯 일곱 여덟 아홉 열 |
| 用于：日期、金钱、电话、分/秒、楼层 | 用于：小时、岁数、个数、杯数 |
| 3시 30분 → 세 시 삼십 분 | 두 잔, 세 개, 다섯 명 |

### 8.2 常用量词

| 量词 | 意义 | 例 |
|-----|------|---|
| 개 | 个 | 사과 두 개 |
| 명 / 분 (敬) | 位/人 | 세 명 / 두 분 |
| 잔 | 杯 | 커피 한 잔 |
| 병 | 瓶 | 물 한 병 |
| 그릇 | 碗 | 밥 한 그릇 |
| 마리 | 只（动物） | 강아지 한 마리 |
| 권 | 本 | 책 한 권 |
| 장 | 张 | 종이 한 장 |

### 8.3 变形注意

- 하나 → 한 (개), 둘 → 두 (잔), 셋 → 세, 넷 → 네, 스물 → 스무 (살)

## Part 9 · 否定

- **안 + V**（前置）= 不
  - 안 먹어요.
- **-지 않다**（后置）= 不
  - 먹지 않아요.
- **못 + V** = 不能（能力/条件不允许）
  - 못 가요.
- **-지 못하다** = 不能
- **-지 마세요** = 请不要

## Part 10 · 高频陷阱 & 中韩差异

1. 助词错用是中文母语者最常见的错误 → 见 common-mistakes 知识库。
2. 语序：韩语是 **SOV**（主 + 宾 + 谓）：나는 밥을 먹어요.
3. 主语常省略：韩语上下文清楚时省略主语，中文人容易多加「저는」。
4. 汉字词直译陷阱：并非所有中文汉字词都能直译，如「爱人 (爱人)」在韩语「애인」= 恋人。
5. 「时间点」与「时间段」助词不同：3시에 (3点整) vs 3시간 동안 (3小时期间)。
