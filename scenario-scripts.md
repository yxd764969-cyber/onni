# Onni 场景对话库（scenario-scripts）

> 四大场景真人对话脚本，Coze 场景对话节点检索用。每段含：场景说明 → 关键句 → 完整对话 → 常见变体。

---

## 场景 1 · 咖啡店点单 (카페에서)

### 1.1 基础点单

**关键句**：
- 아이스 아메리카노 한 잔 주세요. (请给我一杯冰美式。)
- 여기서 드세요? / 포장이세요? (店内喝还是打包？)
- 카드로 결제할게요. (我用卡付。)

**完整对话**：

```
직원: 안녕하세요, 주문 도와드릴게요.
     (您好，为您点单。)
손님: 아이스 아메리카노 한 잔 주세요.
     (请给我一杯冰美式。)
직원: 사이즈는 어떻게 해 드릴까요?
     (要什么尺寸？)
손님: 톨 사이즈로 주세요.
     (要 tall 号。)
직원: 매장에서 드시나요, 포장하시나요?
     (店内喝还是打包？)
손님: 포장이요. (打包。)
직원: 총 4,500원입니다. 결제 도와드릴게요.
     (一共 4500 韩元，为您结账。)
손님: 카드로 할게요. (刷卡。)
```

### 1.2 加料 · 少糖 · 常见变体

- 시럽 빼 주세요. (请去糖浆。)
- 얼음 조금만 넣어 주세요. (请少放冰。)
- 우유 대신 두유로 바꿔 주세요. (请把牛奶换成豆奶。)
- 샷 추가해 주세요. (加一份 shot。)
- 덜 달게 해 주세요. (请少甜。)
- 뜨겁게 / 차갑게 해 주세요. (请做热的/凉的。)

### 1.3 韩国人常用缩写

- **아아** = 아이스 아메리카노
- **뜨아** = 뜨거운 아메리카노
- **따아** = 따뜻한 아메리카노
- **아라** = 아이스 라떼
- 「아아 한 잔이요.」= 一杯冰美 (超口语)

## 场景 2 · 餐厅点菜 (식당에서)

### 2.1 找位子 → 点菜 → 结账

```
직원: 몇 분이세요? (几位？)
손님: 두 명이요. (两位。)
직원: 이쪽으로 오세요. (这边请。)
     주문하시겠어요? (可以点单了吗？)
손님: 김치찌개 하나랑 비빔밥 하나 주세요.
     (泡菜汤一份、拌饭一份。)
직원: 매운맛 괜찮으세요? (辣的可以吗？)
손님: 네, 괜찮아요. 근데 좀 덜 맵게 해 주세요.
     (可以，但请稍微不要那么辣。)
직원: 알겠습니다. 잠시만 기다려 주세요.
     (好的，请稍等。)
--- 用餐后 ---
손님: 저기요, 계산할게요. (服务员，我要结账。)
직원: 총 25,000원입니다. (一共 25000 韩元。)
손님: 따로 계산해 주세요. (请分开结算。)
```

### 2.2 高频请求

- 반찬 좀 더 주세요. (请再给点小菜。)
- 물 좀 주세요. (请给点水。)
- 앞접시 하나 더 주세요. (请再给一个小盘子。)
- 이거 포장 되나요? (这个能打包吗？)
- 남은 거 싸 주세요. (剩下的请打包。)

### 2.3 特殊需求

- 저는 매운 거 못 먹어요. (我不能吃辣。)
- 견과류 알레르기가 있어요. (我对坚果过敏。)
- 고기 빼 주세요. (请去肉。)
- 채식주의자예요. (我是素食者。)

## 场景 3 · 便利店 (편의점에서)

```
손님: 이거 하나 주세요. (这个一个。)
직원: 봉투 필요하세요? (要袋子吗？)
손님: 네, 하나 주세요. 얼마예요?
     (要一个，多少钱？)
직원: 100원입니다. 총 3,500원이에요.
     (100 韩元。一共 3500 韩元。)
손님: 카드요. (刷卡。)
직원: 영수증 드릴까요? (需要小票吗？)
손님: 아니요, 괜찮아요. (不用了，谢谢。)
```

**高频**：
- 데워 드릴까요? (需要加热吗？)
- 네, 데워 주세요. (是的，请加热。)
- 젓가락 하나 주세요. (给一双筷子。)
- 삼각김밥 어디 있어요? (三角饭团在哪儿？)

## 场景 4 · 购物 / 试衣间 / 砍价

### 4.1 服装店

```
손님: 이거 입어 봐도 돼요? (这个可以试穿吗？)
직원: 그럼요, 사이즈 뭐 드릴까요? (当然，要什么尺寸？)
손님: 엠(M) 사이즈로 주세요. (要 M 号。)
--- 试穿后 ---
손님: 조금 커요. 스몰 사이즈 있어요?
     (有点大，有 S 号吗？)
직원: 잠시만요, 확인해 드릴게요. (稍等，我查一下。)
손님: 이거 얼마예요? (这个多少钱？)
직원: 55,000원입니다. (55000 韩元。)
손님: 좀 깎아 주세요. (便宜点儿吧。)
     * 大型商场不砍价，仅限传统市场
```

### 4.2 退换货

- 이거 환불하고 싶어요. (我想退货。)
- 사이즈 교환할 수 있나요? (可以换尺寸吗？)
- 영수증 있어요. (我有小票。)

### 4.3 免税店

- 여권 보여 주세요. (请出示护照。)
- 면세 되나요? (可以免税吗？)

## 场景 5 · 交通出行

### 5.1 地铁

```
손님: 저기요, 이거 강남역 가는 거 맞아요?
     (请问，这个是去江南站的吗？)
행인: 네, 맞아요. 세 정거장이에요.
     (是的，三站。)
손님: 몇 호선으로 갈아타야 해요?
     (要换乘几号线？)
행인: 2호선에서 3호선으로 갈아타세요.
     (从 2 号线换 3 号线。)
```

**必备**：
- 여기가 어디예요? (这是哪儿？)
- ~ 가는 길 좀 알려 주세요. (请告诉我去 ~ 的路。)
- 몇 번 출구로 나가야 해요? (要从几号出口出去？)
- 티머니 카드 어디서 사요? (T-money 卡在哪儿买？)

### 5.2 出租车

```
손님: 인천공항까지 얼마나 걸려요?
     (到仁川机场要多久？)
기사: 한 시간쯤 걸려요. (大概一小时。)
손님: 좀 빨리 가 주세요. 시간이 없어요.
     (请开快点，我时间不多。)
기사: 네, 알겠습니다. (好的。)
--- 到达 ---
손님: 여기서 세워 주세요. (在这儿停。)
     카드 되나요? (可以刷卡吗？)
```

**必备**：
- 이 주소로 가 주세요. (请去这个地址。)
- 미터기로 가 주세요. (请打表。)
- 어디에서 세워 드릴까요? (在哪儿停？)

### 5.3 问路

- 화장실이 어디예요? (洗手间在哪儿？)
- 편의점이 근처에 있어요? (附近有便利店吗？)
- 여기서 얼마나 걸어야 해요? (从这儿要走多久？)

## 场景 6 · 初次见面 / 社交

### 6.1 自我介绍

```
A: 안녕하세요, 처음 뵙겠습니다.
   (您好，初次见面。)
B: 안녕하세요, 반갑습니다.
   (您好，很高兴认识您。)
A: 저는 왕리라고 합니다. 중국에서 왔어요.
   (我叫王丽，从中国来。)
B: 아, 그러세요? 한국어 잘하시네요.
   (啊，是吗？韩语说得真好。)
A: 아직 부족해요. 잘 부탁드립니다.
   (还不够好，请多关照。)
```

### 6.2 加联系方式

- 카톡 아이디 뭐예요? (KakaoTalk ID 是什么？)
- 전화번호 알려 주실 수 있어요? (可以告诉我电话号码吗？)
- 인스타 하세요? (玩 Instagram 吗？)

### 6.3 约见面

```
A: 이번 주말에 시간 있어요?
   (这周末有空吗？)
B: 네, 토요일이 좋아요. 뭐 할까요?
   (有的，周六可以，做什么呢？)
A: 홍대에서 밥 먹고 카페 갈까요?
   (在弘大吃个饭再去咖啡店怎么样？)
B: 좋아요! 몇 시에 만날까요?
   (好啊，几点见？)
A: 12시 어때요? 홍대입구역 9번 출구에서 만나요.
   (12点怎么样？在弘大入口站 9 号出口见。)
```

## 场景 7 · 医院 / 药房

### 7.1 挂号 · 描述症状

```
접수: 처음 오셨어요? (第一次来吗？)
환자: 네, 예약 안 했는데 진료 볼 수 있어요?
     (是的，没预约能看诊吗？)
접수: 신분증 주세요. 어디가 불편하세요?
     (请给身份证。哪里不舒服？)
환자: 어제부터 목이 아프고 열이 나요.
     (从昨天开始嗓子疼、发烧。)
```

**症状常用表达**：
- 머리가 아파요. (头痛。)
- 배가 아파요. (肚子痛。)
- 열이 나요. (发烧。)
- 기침이 나요. (咳嗽。)
- 콧물이 나요. (流鼻涕。)
- 어지러워요. (头晕。)
- 토할 것 같아요. (想吐。)
- 알레르기가 있어요. (有过敏。)

### 7.2 药房

```
손님: 감기약 있어요? (有感冒药吗？)
약사: 어떤 증상이세요? (什么症状？)
손님: 기침이 심해요. (咳嗽厉害。)
약사: 이 약 드셔 보세요. 하루 세 번, 식후에 드세요.
     (吃这个药，一天三次，饭后吃。)
```

## 场景 8 · 美容院 / 美甲 / 皮肤管理

### 8.1 预约

```
손님: 예약하려고 전화 드렸어요.
     (打电话是想预约。)
직원: 어떤 시술 원하세요? (要做什么项目？)
손님: 젤 네일이랑 페디큐어요. (光疗美甲和足部护理。)
직원: 언제 오실 수 있으세요? (什么时候可以来？)
손님: 이번 주 금요일 오후 3시 가능해요?
     (这周五下午 3 点可以吗？)
```

### 8.2 描述需求

- 이 사진처럼 해 주세요. (请照这张照片做。)
- 좀 자연스럽게 해 주세요. (请自然一点。)
- 너무 진하지 않게 해 주세요. (请不要太浓。)
- 이 색깔로 해 주세요. (请做这个颜色。)
- 앞머리는 자르지 말아 주세요. (请不要剪刘海。)

### 8.3 美发店

- 다듬어 주세요. (请修一下。)
- 5cm 정도 잘라 주세요. (剪 5 厘米左右。)
- 파마해 주세요. (烫发。)
- 염색하고 싶어요. (想染发。)

## 场景 9 · 机场 / 值机

```
직원: 여권 보여 주시겠어요? (请出示护照。)
승객: 여기 있어요. (在这儿。)
직원: 창가 자리, 통로 자리 중 어디로 드릴까요?
     (要靠窗还是靠过道？)
승객: 창가 자리로 부탁드려요. (靠窗。)
직원: 부칠 짐 있으세요? (有托运行李吗？)
승객: 캐리어 하나 있어요. (有一个行李箱。)
```
