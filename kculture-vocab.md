# Onni K-culture 词汇库（kculture-vocab）

## 一、饮食文化

### 咖啡缩写文化
- **아아** = 아이스 아메리카노（冰美式）
- **뜨아** = 뜨거운 아메리카노（热美式）
- **따아** = 따뜻한 아메리카노（温美式）
- **아라** = 아이스 라떼
- **얼죽아** = 얼어 죽어도 아이스 아메리카노（冻死也要喝冰美）

### 独食 / 独饮文化
- **혼밥** = 혼자 밥 먹기（一个人吃饭）
- **혼술** = 혼자 술 마시기（一个人喝酒）
- **혼영** = 혼자 영화 보기（一个人看电影）
- **혼카** = 혼자 카페 가기（一个人去咖啡店）

### 韩国国民食
- 김치찌개 (泡菜汤)
- 된장찌개 (大酱汤)
- 순두부찌개 (嫩豆腐汤)
- 부대찌개 (部队锅)
- 삼겹살 (五花肉烤肉)
- 치맥 (chi-maek) = 치킨 + 맥주（炸鸡+啤酒）
- 소맥 (so-maek) = 소주 + 맥주（烧酒+啤酒）
- 라면 (拉面) / 컵라면 (杯面)
- 떡볶이 (辣炒年糕)
- 순대 (糯米血肠)
- 김밥 (紫菜卷饭)
- 삼각김밥 (三角饭团)
- 붕어빵 (鲷鱼烧)

### 网红 / 排队
- 맛집 [mat-jip] = 好吃的店（网红美食）
- 핫플 [hat-peul] = hot place，网红地
- 웨이팅 [we-i-ting] = 排队
- 오픈런 [o-peun-reon] = 开店冲刺（限量商品开门就冲）

## 二、K-pop / 追星文化

### 基础
- **최애** [choe-ae] = 最爱（本命）
  - 예: 제 최애는 지민이에요.
- **차애** [cha-ae] = 第二喜欢
- **덕질** [deok-jjil] = 追星、饭圈活动
- **덕후** [deok-hu] = 死忠饭
- **입덕** [ip-deok] = 入坑
- **탈덕** [tal-deok] = 脱粉
- **성덕** [seong-deok] = 成功的粉丝（见到偶像/合作）

### 演唱会 / 应援
- **콘서트** [kon-seo-teu] = 演唱会
- **팬미팅** [paen-mi-ting] = 粉丝见面会
- **팬싸** [paen-ssa] = 팬사인회（签售会）
- **응원봉** [eung-won-bong] = 应援棒
- **떼창** [tte-chang] = 全场大合唱
- **직캠** [jik-kaem] = 饭拍
- **음방** [eum-bang] = 音乐节目

### 团/成员术语
- **아이돌** [a-i-dol] = 偶像
- **멤버** [mem-beo] = 成员
- **리더** [ri-deo] = 队长
- **막내** [mang-nae] = 老幺
- **비주얼** [bi-ju-eol] = 门面
- **센터** [sen-teo] = C 位
- **컴백** [keom-baek] = 回归（发新专辑）
- **활동** [hwal-dong] = 活动期

### 感叹 / 反应
- **미쳤다** [mi-cheot-tta] = 疯了（超绝、超好，褒义感叹）
- **찢었다** [jji-jeot-tta] = 撕了（表演炸裂）
- **레전드** [re-jeon-deu] = legend（名场面）
- **소름** [so-reum] = 起鸡皮疙瘩

## 四、网络 / 年轻人流行语

- **핵** [haek] = 超级（前缀）
  - 예: 핵맛있다 (超好吃), 핵꿀잼 (超有趣)
- **꿀잼** [kkul-jaem] = 蜂蜜有趣 = 超有趣
- **노잼** [no-jaem] = no + 有趣 = 无聊
- **인싸** [in-ssa] = insider，社交达人
- **아싸** [a-ssa] = outsider，边缘人
- **JMT** = 존맛탱 = 超级好吃
- **갓** [gat] = God + 名词，膜拜级
- **띵작** [tting-jak] = 名作（网络体）
- **개** [gae] = 超（俚俗，注意场合）
  - 예: 개웃겨 (超好笑)
- **ㅋㅋㅋ** = 哈哈哈（韩国网友笑声）
- **ㅠㅠ / ㅜㅜ** = 呜呜（哭）
- **오키** = OK
- **넵** [nep] = 是的（工作聊天用，比 네 更活泼）
