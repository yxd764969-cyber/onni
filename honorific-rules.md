# Onni 敬语规则库（honorific-rules）

> Coze 检索用：判断用户敬语等级、纠错、给出建议时优先调用本库。

---

## 一、韩语文体三大等级（一句话记住）

| 等级 | 语尾 | 例：吃饭 | 使用场合 |
|-----|------|---------|---------|
| 합쇼체（格式体） | -ㅂ니다/습니다 | 밥을 먹습니다 | 新闻、演讲、军队、面试、非常正式的商务 |
| 해요체（非格式敬语） | -아요/어요 | 밥을 먹어요 | **日常礼貌默认**：店员、陌生人、长辈、同事 |
| 해체 / 반말（平语） | -아/어 | 밥 먹어 | 亲近朋友、恋人、比自己小、家人间 |

**Onni 教学核心默认：해요体**（日常最实用）。

---

## 二、场景 → 用哪个体？

### 必用 해요体（略敬）：
- ☕ 咖啡店、餐厅、便利店、商店（对店员）
- 🚇 出租车司机、地铁行人
- 👋 初次见面（无论对方年龄）
- 🏥 医院、药房、美容院
- 👨‍🏫 老师、上司（同时可能配合 (으)시-）
- 📞 客服电话

### 可用 반말（平语）：
- 🧑‍🤝‍🧑 关系熟悉的同龄朋友（**双方明确同意后**）
- 👨‍👩‍👧 家人之间（对小辈）
- 💑 恋人
- 👶 对小孩

### 必用 합쇼体：
- 📺 主播、播报、演讲
- 🎤 面试、商务谈判、军队
- 📝 正式书面（报告、公文）

## 三、如何"升级"到敬语？三层加码

### 第 1 层：语尾变为 해요体
- 먹어 → 먹어요 / 가 → 가요

### 第 2 层：主语敬语 -(으)시-
对**长辈、上司、客户**加 시，表示尊敬对方这个人。
- 아버지가 오세요. (爸爸来。← 시 尊敬爸爸)
- 사장님이 말씀하세요. (社长在说话。)

**结构**：词干 + (으)시 + 语尾 → 오시 + 어요 = 오세요

### 第 3 层：敬语专用词汇替换（最高级）
某些高频词有专用敬语版本，替换效果比加 시 更礼貌。

| 一般 | 敬语 | 意义 |
|-----|------|-----|
| 있다 (在) | 계시다 | 在（人） |
| 있다 (有) | 있으시다 | 有 |
| 먹다 / 마시다 | 드시다 / 잡수시다 | 吃 / 喝 |
| 자다 | 주무시다 | 睡 |
| 죽다 | 돌아가시다 | 去世 |
| 말하다 | 말씀하시다 | 说 |
| 주다（给对方） | 드리다 | 给（自谦） |
| 만나다 | 뵙다 / 뵈다 | 见 |
| 묻다 | 여쭙다 / 여쭈다 | 问（自谦） |
| 데리고 가다 | 모시고 가다 | 带（长辈）去 |

**名词类**：
| 一般 | 敬语 | 意义 |
|-----|------|-----|
| 이름 | 성함 | 姓名 |
| 나이 | 연세 / 춘추 | 年龄 |
| 생일 | 생신 | 生日 |
| 집 | 댁 | 家 |
| 밥 | 진지 | 饭 |
| 사람 | 분 | 人 |
| 말 | 말씀 | 话 |

**助词类**：
| 一般 | 敬语 | 意义 |
|-----|------|-----|
| 이/가 | 께서 | 主语（对长辈） |
| 은/는 | 께서는 | 主题（对长辈） |
| 에게/한테 | 께 | 给…（长辈） |

## 四、"我"的自谦

- 나 → 저（正式/敬语场合用 저）
- 우리 → 저희

**易错**：初次见面用「나는」= 不礼貌，应用「저는」。

## 五、常见错误判定

### ❌ 错误 1：对陌生人用 반말
```
错：여기 커피 한 잔.
对：여기 커피 한 잔 주세요.
理由：对店员必须 해요体+주세요。
```

### ❌ 错误 2：对长辈用 나
```
错：나는 학생이에요.（对老师）
对：저는 학생이에요.
理由：对长辈自称必须用 저。
```

### ❌ 错误 3：对长辈忘了加 시
```
错：할머니가 밥 먹어요.
对：할머니께서 진지 드세요.
理由：对奶奶要用 께서 + 진지 + 드시다。
```

### ❌ 错误 4：过度使用 합쇼体
```
过：저는 학생입니다. 오늘 커피 한 잔 마시겠습니다.（跟朋友说）
对：나 오늘 커피 한 잔 마셨어.
理由：合쇼体只用在非常正式场合，日常会显得很生硬。
```

### ❌ 错误 5：混用文体
```
错：저는 왔어요. 배고파.（一句敬语一句平语）
对：저는 왔어요. 배고파요.
理由：一次对话中文体必须统一。
```

## 六、Onni 纠错模板

用户韩语敬语等级不对时，Onni 用以下模板回复：

```
🌸 温柔敬语提示
你说的：[用户原句]
建议改成：[修正后的句子]

理由：[场景 + 应用的敬语等级]
比如现在是【咖啡店点单】场景，对店员要用【해요体 + 주세요】更礼貌哦～

小口诀：
"陌生人 = 해요体"
"长辈 = 해요体 + 시 + 敬语词"
```

## 七、Onni 快速判断流程（决策树）

```
用户对谁说话？
│
├─ 陌生人 / 服务员 / 初次见面 → 해요体（默认）
│
├─ 长辈 / 上司 / 客户 → 해요体 + (으)시 +（可能）敬语词
│
├─ 特别正式（面试/演讲/长辈+）→ 합쇼体 + (으)시 + 敬语词
│
└─ 熟悉同龄朋友 / 家人小辈 / 恋人 → 반말
```

## 八、场景速查（Onni 输出建议时对照）

| 用户在什么场景 | Onni 应该教哪种体 | 常用句尾示例 |
|--------------|-----------------|-------------|
| 咖啡店 / 便利店 / 餐厅 | 해요体 | 주세요, 얼마예요?, 있어요? |
| 出租车 / 地铁 | 해요体 | 가 주세요, 세워 주세요 |
| 医院 / 药房 | 해요体（对医生） | 아파요, 드세요 |
| 美容院 | 해요体 | 해 주세요, 이렇게요 |
| 初次社交 | 해요体 | 이에요, 반갑습니다 |
| 加联系方式 | 해요体 | 카톡 아이디 뭐예요? |
| 熟朋友闲聊 | 반말 | 뭐 해?, 어디야? |
| 面试 / 商务 | 합쇼体 | 감사합니다, 이해했습니다 |
